Giftsmith app icon set

Source: isolated from the selected monogram concept.
Palette: dark forest canvas with brass mark.

iOS:
- Use ios/AppIcon.appiconset in Xcode Assets.xcassets.
- Icons are exported without alpha, as required by App Store.

Android:
- Copy android/app/src/main/res/* into your Android app res folder.
- Includes legacy mipmap PNGs and adaptive icon XML scaffold.

Note:
The source artwork was isolated from the generated mockup, so this is suitable for app prototyping and store-preview work. For final production, recreate the mark as vector artwork and export from the vector master.
