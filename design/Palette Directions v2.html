Both directions ship as a light + dark pair, with the system color (Auto) honored by default. Theme B in each is the secondary mode — it's intentionally not a literal inversion; the mood travels but the surfaces are tuned for that environment.

A note on auto-switching

On iOS, the app should follow system appearance by default with a manual override in settings. Both directions are designed so neither theme feels like the "second-class" version — Ember works best with Linen-light as default; Midnight Garden works best with Midnight-dark as default.
