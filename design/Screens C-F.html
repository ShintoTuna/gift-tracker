Phase 03 · Midnight Garden · Screens C–F

People · Calendar · Backlog · Brainstorm.

Four screens in the locked language. People is sorted by next occasion with a "This week" urgency group above the rest. Calendar groups by month with a top stats strip. Backlog uses filter chips with one active filter shown. Brainstorm leads with a context block (interests, budget, avoiding) and lists 5 numbered suggestions, each with reasoning + price + Save action — explicitly not a chat.

Where the language strained

Bent rules

  • Avatar stack on Backlog cards. Two overlapping 22px circles work but get muddy at 3+. Cap at 2 + "+N" pill, or rethink for ideas tagged to many people.
  • Filter chips horizontally scrolling. Off-screen filters are hard to discover. Might need a "Filters · 3" entry that opens a sheet once we have more than four facets.
  • Date column on Calendar rows. The little day/weekday block is technically a fourth visual element competing with the avatar/icon vocabulary. I dropped person avatars on this screen to compensate — a tradeoff worth validating.
  • Brainstorm "01–05" mono numerals work but the screen has no other numbered list in the system. If we use ordering elsewhere it'll feel consistent; if not, it stands out.
Worth designing next

States to validate

  • People — empty. First-run, no people added. Crucial moment; should sell the share-sheet capture concept.
  • Calendar — quiet stretch. What does the "next occasion is 4 months away" view feel like? The stats strip and dense rows assume a busy life.
  • Brainstorm — generating. The 3–8 second loading state. Skeleton cards or a "thinking…" copy moment is a chance to set tone.
  • Brainstorm — error / context too thin. "I don't have enough about Eleanor yet — add 2 interests?" — failure mode that turns into onboarding.
New components → back-port

Reuse in v2 screens

  • Avatar atom (3 sizes). Standardize 22 / 32 / 44 / 56 for stack / row / list / hero. Apply to Profile header.
  • Eyebrow component with optional accent color (brass for "Next occasion", claret for urgency). Already used everywhere; codify.
  • StatPill row from Calendar could appear at the top of Profile too: "next · 18d · 4 ideas · last gift May '25".
  • Filter chip atom (active / dashed / default) → use for Profile interests & the typeahead "+ Someone else" in Quick Capture.
  • Suggestion card pattern (mono index + italic title + reasoning + footer with price & Save) → reuse for any AI moment in the app.
